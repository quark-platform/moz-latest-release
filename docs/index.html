Mozilla latest release api

This API returns information regarding the latest releases of a variety of Firefox products and their source code properties. APIs that are available are

  • /version/:target - Returns the latest version at this target
  • /source/:target - Redirects to the download for the latest source code of this target
  • /source/:target/url - The url to download the latest source code of this target

Notes

  • A list of :targets can be found by entering an incorrect value
